## ğŸ¯ Customization

### Change Company Name
Replace `TechVision` throughout:
- HTML: Logo and titles
- CSS: Color schemes
- JavaScript: Console message

### Update Colors
Modify CSS variables in `style.css`:
```css
:root {
    --primary-color: #0066cc;
    --accent-cyan: #00d4ff;
    /* ... more colors ... */
}
```

### Add Your Logo
Add an `<img>` tag in the navbar:
```html
<img src="assets/images/logo.png" alt="Company Logo" class="logo-img">
```

### Update Contact Information
Edit footer and contact section:
- Phone number
- Email address
- Office location
- Social media links
