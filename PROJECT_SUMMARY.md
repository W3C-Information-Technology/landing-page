## ğŸ¯ Customization

### 1. Change Company Information
Find in `index.html` and update:
- Company name (search "TechVision")
- Contact info (phone, email, address)
- Services and descriptions
- Portfolio projects
- Testimonials

### 2. Update Colors
Edit `:root` variables in `assets/css/style.css`:
```css
--primary-color: #0066cc;      /* Change to your brand color */
--accent-cyan: #00d4ff;        /* Accent color */
--accent-purple: #9333ea;      /* Secondary accent */
```

### 3. Add Your Logo
Replace the text logo in navbar with an image:
```html
<img src="assets/images/logo.png" alt="Your Logo" class="logo-img">
```

### 4. Add Images
Place images in `assets/images/` folder and update paths in HTML
